# icinga2-check_ucopia

icinga2-check_ucopia is an icinga2 plugin to check the status of ucopia controllers

The plugin uses snmp.

The following services can be monitored:
- autodisconnectService
- cpuTemperature
- dhcpServerService
- diskTemperature
- dnsServerService
- highAvailabilityService
- ldapDirectoryService
- ldapReplicationManagerService
- licensedUsers
- licensesleft
- pmsClientService
- portalService
- printingServerService
- radiusServerService
- sambaService
- sqlServerService
- sshService
- staticIpManagerService
- syslogService
- timeServerService
- totalConnectedUsers
- urlSnifferService
- usersLogService
- webProxyService
- webServerService
